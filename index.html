Convite para o Hackathon Faculdades Rio Branco Granja Viana

O que é um hackathon?

Um hackathon é um evento intensivo — geralmente com duração de 24 a 72 horas — onde programadores, designers, analistas e outros profissionais da tecnologia se reúnem para criar soluções de software ou hardware em um curto período de tempo.

Seguindo essa lógica, o hackathon Faculdades Rio Branco Granja Viana tem o objetivo de promover a participação de alunos em um projeto de desenvolvimento de software para aplicar seus conhecimentos em uma dinâmica rápida, e também um trabalho em equipe a fim de ser usado como base de raciocínio lógico e aplicação técnica para resolução de problemas simples do cotidiano.

Quero participar!

1. Níveis de Dificuldade por Tamanho de Grupo

  • Duplas: 4 horas de desenvolvimento.
  • Trios: 3 horas de desenvolvimento.
  • Quartetos: 2 horas de desenvolvimento.

2. Execução

O projeto não precisa ser publicado online. Basta que funcione localmente no computador do grupo, com qualquer stack de tecnologias escolhida.

3. Requisitos Técnicos

O projeto deve conter back-end (obrigatório) e o front-end opcional.

  • Se conter frontend deve se atentar aos principios básicos de UX.
  • Se não conter frontend se atentar à lógica aplicada no backend.

4. Uso de Inteligência Artificial

  • Duplas: uso ilimitado de IA por pessoa.
  • Trios: até 20 prompts por pessoa.
  • Squads: até 10 prompts por pessoa.

O uso será supervisionado por um professor responsável.

5. Avaliação

  • Alunos:Cada alunos irá avaliar o projeto apresentado com uma nota de 0 a 10.
  • No final será feita uma média para chegar no resultado do grupo.
  • Comissão Técnica: professores atribuirão entre 0 e 1 ponto (total) extra com base na qualidade e ideia.
